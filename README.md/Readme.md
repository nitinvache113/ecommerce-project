# E-Commerce Sales Analytics Dashboard

## Project Overview

This project analyzes E-Commerce sales data using Python, Power BI, and Data Analytics techniques to generate actionable business insights.

## Objectives

* Analyze sales performance
* Identify profit trends
* Understand customer purchasing behavior
* Visualize key business metrics
* Support data-driven decision making

## Tools & Technologies

* Python
* Pandas
* NumPy
* Power BI
* Excel
* CSV Dataset
